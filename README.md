# Techsheets-api

## Techsheets Api

## Project description

With Techsheets api you can manage you own technical sheets for your pastry work.
Helps you by registering your Ingredients with prices your recipes and multiple recipes gives you a technical pastry sheet.

## Who this project is for

This project is intended to work with a this [front end](https://github.com/elkofy/fiche-patisserie) but you could use it with your own.

## Project dependencies

Before using Techsheets api , ensure you have thes installed:

* [Go](https://go.dev/doc/tutorial/getting-started)
* [Air](https://github.com/air-verse/air)
